# Space

Add some space to components

<!-- example -->
```jsx
import Space from 'react-ua/lib/elements/space';

class SpaceTest extends React.Component {

  render () {
    return (
      <Space w={12}/>
    );
  }
}
```

## Properties

| Name              | Type          | Default         | Description |
|:-----|:-----|:-----|:-----|
| `w`        | `Number`       |  1        | Increase space .|
