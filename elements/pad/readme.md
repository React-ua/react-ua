# Pad

Add some padding to components

<!-- example -->
```jsx
import Pad from 'react-ua/lib/elements/pad';

class PadTest extends React.Component {

  render () {
    return (
      <Pad x={12} y={12}/>
    );
  }
}
```

## Properties

| Name              | Type          | Default         | Description |
|:-----|:-----|:-----|:-----|
| `x`        | `Number`       |  1        | Increase padding .|
| `y`        | `Number`       |  1        | Increase padding .|
