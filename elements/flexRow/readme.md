# FlexRow

Container set to display flex with gutters

<!-- example -->
```jsx
import FlexRow from 'react-ua/lib/elements/flexRow';

class FlexRowTest extends React.Component {

  render () {
    return (

      <FlexRow >

        <t>
        Helloooo
        </t>
        <t>
        Helloooo
        </t>

      </FlexRow>

      );
    }
  }
  ```
