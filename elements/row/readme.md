# Row

Row element have a Clearfix and negative-margin gutter compensation

<!-- example -->
```jsx
import Row from 'react-ua/lib/elements/row';

class RowTest extends React.Component {

  render () {
    return (
      <Row>
      <div sm={6}>
      Col 6
      </div>
      <div sm={6}>
      Col 6
      </div>
      </Row>
    );
  }
}
```
